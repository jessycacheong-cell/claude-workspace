- Status: Planned

### What it is
A personal command center — one dark-mode page with everything in one place.

### Sections to build
- Workspace Map — visual of claude-workspace structure (brands/apps/personal/templates)
- Active Projects — milestones and status for each brand/app
- Brand Quick Cards — click a brand to see colors, contacts, tone
- GitHub Links — quick access to all repos
- Social & Calendar — upcoming posts and events
- Task Manager — add/edit/delete tasks with priority levels, auto-runs hourly via cron job, logs activity to tasks.log

### Design Rules
- Dark mode, UMEI brand aesthetic (deep space background, daydream blue accents)
- Mobile-first
- Single HTML file, no framework
- Interactive — clickable cards, expandable sections
